Barndoor

BarnDoor provides registered users with the ability to create and manage livestock data in one easy-to-use app. The application keeps track of the name, ear tag number, gender, breed, pedigree, herd name, date of birth, weight, and vaccine records of an individual animal. HTML, CSS, Bootstrap, JQuery, Javascript, Spring, Thymeleaf, Java, MySQL, Filestack API, and a Treant Javascript library.
